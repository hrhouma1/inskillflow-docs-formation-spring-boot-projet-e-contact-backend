## Outils de developpement

### IDE

- **IntelliJ IDEA**: https://www.jetbrains.com/idea/
  - Community (gratuit) ou Ultimate
  - Meilleur support Spring

- **Visual Studio Code**: https://code.visualstudio.com/
  - Gratuit, leger
  - Extensions: Java Extension Pack, Spring Boot Extension Pack

### Tests d'API

- **Postman**: https://www.postman.com/
  - Client HTTP complet
  - Collections, environments

- **Insomnia**: https://insomnia.rest/
  - Alternative a Postman

- **curl**: Ligne de commande
  - Inclus dans Git Bash (Windows)

### Base de donnees

- **DBeaver**: https://dbeaver.io/
  - Client universel gratuit
  - Supporte PostgreSQL, H2, etc.

- **pgAdmin**: https://www.pgadmin.org/
  - Client specifique PostgreSQL

### Monitoring

- **Docker Desktop**: https://www.docker.com/products/docker-desktop
  - Interface graphique Docker

- **Portainer**: https://www.portainer.io/
  - UI web pour Docker
